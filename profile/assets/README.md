# assets

`nebelhaus-banner-rounded.png` — the banner at the top of the org profile, and
the single most important file in this repo. `profile/README.md` references it
by raw GitHub URL, so keep the filename stable when you replace it.

The org profile lives or dies on this one image. It should carry, in one frame:

- **Pounce** open (the command palette — proves it's a real native app)
- **Nebelung** colours obvious across at least two apps (the silver-grey identity)
- **SketchyBar** perched at the top edge
- **AeroSpace** tiling visible (a couple of tiled windows)

Aim for a clean desktop, a real workflow on screen (not an empty terminal), and
retina resolution. To swap it, drop the new render in over the same filename. A
short screen recording as a follow-up is a strong bonus, but the still comes
first — and if you add one, reference it explicitly from `profile/README.md`.
