# assets

Drop `hero.png` here — the single most important file in this repo.

The org profile lives or dies on this one screenshot. It should show, in one frame:

- **Pounce** open (the command palette — proves it's a real native app)
- **Nebelung** colours obvious across at least two apps (the silver-grey identity)
- **Sill** perched at the top edge (SketchyBar)
- **Prowl** tiling visible (a couple of tiled windows)

Aim for a clean desktop, a real workflow on screen (not an empty terminal),
and retina resolution. A 15–20s screen recording (`hero.mp4` / `.gif`) as a
follow-up is a strong bonus but the still comes first.
